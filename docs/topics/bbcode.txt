[B][COLOR=#0099ff]⚡ Commands[/COLOR][/B]
[LIST]
[*][B]/balance[/B]: View your balance
[*][B]/balance <player>[/B]: View another player's balance (ezeconomy.balance.others)
[*][B]/eco <give|take|set> <player> <amount>[/B]: Administrative balance controls (ezeconomy.eco)
[*][B]/eco gui[/B]: Show balance GUI
[*][B]/baltop [amount][/B]: Show top balances
[*][B]/bank <create|delete|balance|deposit|withdraw|addmember|removemember|info> ...[/B]: Bank management (ezeconomy.bank.*)
[*][B]/pay <player> <amount>[/B]: Send funds to another player (ezeconomy.pay)
[*][B]/currency [currency][/B]: Set or view your preferred currency (ezeconomy.currency)
[*][B]/ezeconomy cleanup[/B]: Remove orphaned player data (ezeconomy.admin)
[*][B]/ezeconomy daily reset[/B]: Reset all daily rewards (ezeconomy.admin)
[*][B]/ezeconomy reload[/B]: Reload plugin configuration (ezeconomy.admin)
[*][B]/ezeconomy reload messages[/B]: Reload only the message file (ezeconomy.admin)
[*][B]/ezeconomy database info[/B]: Show database connection info (ezeconomy.admin)
[*][B]/ezeconomy database test[/B]: Test the database connection (ezeconomy.admin)
[*][B]/ezeconomy database reset[/B]: Reset all database tables (DANGEROUS) (ezeconomy.admin)
[*][B]/tax[/B]: Removed — tax functionality moved to EzTax (https://modrinth.com/plugin/eztax)
[/LIST]

[B][COLOR=#0099ff] Permissions[/COLOR][/B]
[LIST]
[*]ezeconomy.balance.others: View other players' balances
[*]ezeconomy.eco: Use /eco admin command
[*]ezeconomy.pay: Use /pay command
[*]ezeconomy.currency: Use /currency command
[*]ezeconomy.admin: Use /ezeconomy admin commands (cleanup, reload, database, daily reset)
[*]Note: Tax permissions and configuration have been removed from EzEconomy and moved to EzTax (https://modrinth.com/plugin/eztax)

[LIST]

[*]ezeconomy.bank.deposit: Deposit to a bank
[*]ezeconomy.bank.withdraw: Withdraw from a bank
[*]ezeconomy.bank.addmember: Add a member to a bank
[*]ezeconomy.bank.removemember: Remove a member from a bank
[*]ezeconomy.bank.info: View bank info

[/LIST]

[B][COLOR=#0099ff] Security Notes for Server Owners[/COLOR][/B]
[LIST]
[*][B]Limit admin permissions[/B]: Only grant [icode]ezeconomy.eco[/icode] and [icode]ezeconomy.bank.admin[/icode] to trusted staff. Tax management is handled by EzTax.
[*][B]Use a permissions plugin[/B]: Manage access with groups/roles so players cannot self-assign economy powers.
[*][B]Lock down database access[/B]: Use a dedicated database user with minimal privileges and keep credentials private.
[*][B]Back up economy data[/B]: Schedule regular backups of your storage files or database to recover from mistakes or exploits.
[*][B]Review bank permissions[/B]: Consider limiting bank creation/withdraw permissions to prevent abuse on public servers.
[/LIST]

[B][COLOR=#0099ff]️ Dupe Prevention Safeguards[/COLOR][/B]
[LIST]
[*][B]Thread-safe balance updates[/B]: Economy operations are designed to avoid race conditions during concurrent deposits, withdrawals, and transfers.
[*][B]Server-side validation[/B]: Commands and transactions validate amounts to prevent invalid or malformed requests.
[*][B]Storage integrity[/B]: Backends use consistent write patterns to reduce the risk of partial or conflicting balance writes.
[/LIST]

[B][COLOR=#0099ff]⚙️ Configuration Example[/COLOR][/B]
[LIST]
[*][B]config.yml[/B] (Only global settings):
[code]
storage: yml
multi-currency:
  enabled: false
  default: "dollar"
  currencies:
    dollar:
      display: "Dollar"
      symbol: "$"
      decimals: 2
    euro:
      display: "Euro"
      symbol: "€"
      decimals: 2
    gem:
      display: "Gem"
      symbol: "♦"
      decimals: 0
  conversion:
    dollar:
      euro: 0.95
      gem: 0.01
    euro:
      dollar: 1.05
      gem: 0.012
    gem:
      dollar: 100
      euro: 80
[/code]
[*][B]config-yml.yml[/B] (YML storage settings):
[code]
yml:
  file: balances.yml
  per-player-file-naming: uuid
  data-folder: data
[/code]
[*][B]config-mysql.yml[/B] (MySQL storage settings):
[code]
mysql:
  host: localhost
  port: 3306
  database: ezeconomy
  username: root
  password: password
  table: balances
[/code]
[*][B]config-sqlite.yml[/B] (SQLite storage settings):
[code]
sqlite:
  file: ezeconomy.db
  table: balances
  banksTable: banks
[/code]
[*][B]config-mongodb.yml[/B] (MongoDB storage settings):
[code]
mongodb:
  uri: mongodb://localhost:27017
  database: ezeconomy
  collection: balances
  banksCollection: banks
[/code]
[/LIST]

[B][COLOR=#0099ff]️ Developer: Custom Storage Providers[/COLOR][/B]
EzEconomy supports custom storage backends (YML, MySQL, SQLite, MongoDB, or your own). You can implement your own provider for any database or storage system.

[B]How to add a custom provider:[/B]
[LIST]
[*]Implement the [icode]StorageProvider[/icode] interface in your plugin or module.
[*]Register your provider before EzEconomy loads:
[code]EzEconomy.registerStorageProvider(new YourProvider(...));[/code]
[*]Only one provider can be registered. If set, EzEconomy will use it instead of YML/MySQL.
[*]See the JavaDoc in [icode]StorageProvider.java[/icode] for required methods.
[/LIST]
